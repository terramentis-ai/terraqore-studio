## üêç Python Script to Split Files

Save this as `split_artifacts.py`:

```python
#!/usr/bin/env python3
import os
import re

def split_artifact_file(artifact_content, base_dir='terraqore-frontend'):
    """Split artifact content into individual files"""
    
    # Pattern to match file headers
    pattern = r'// ={70,}\n// FILE: (.+?)\n// ={70,}\n(.*?)(?=\n// ={70,}|\Z)'
    
    matches = re.findall(pattern, artifact_content, re.DOTALL)
    
    for filepath, content in matches:
        full_path = os.path.join(base_dir, filepath)
        
        # Create directory if it doesn't exist
        os.makedirs(os.path.dirname(full_path), exist_ok=True)
        
        # Write file
        with open(full_path, 'w') as f:
            f.write(content.strip() + '\n')
        
        print(f'‚úì Created: {full_path}')

# Usage:
# 1. Copy artifact content to a file (e.g., 'part1.txt')
# 2. Run: python split_artifacts.py

if __name__ == '__main__':
    import sys
    
    if len(sys.argv) < 2:
        print("Usage: python split_artifacts.py <artifact_file>")
        sys.exit(1)
    
    artifact_file = sys.argv[1]
    
    with open(artifact_file, 'r') as f:
        content = f.read()
    
    split_artifact_file(content)
    print('\n‚úÖ All files created successfully!')
```

**Usage:**
```bash
# Copy Part 1 content to part1.txt
python split_artifacts.py part1.txt

# Repeat for all parts
python split_artifacts.py part2.txt
python split_artifacts.py part3.txt
# ... etc
```

## üé® Customization Tips

### Change Theme Colors
Edit `tailwind.config.js`:
```javascript
theme: {
  extend: {
    colors: {
      primary: '#your-color',
      secondary: '#your-color',
    }
  }
}
```

### Add New Agent Type
1. Update `src/utils/constants.js` - Add to `AGENT_TYPES`
2. Update `src/components/shared/AgentTypeBadge.jsx` - Add icon config
3. Update backend to support new type

### Modify Settings
1. Add new setting to `src/store/settingsStore.js`
2. Create section component in `src/components/settings/`
3. Import and add to `SettingsPanel.jsx`

## üêõ Troubleshooting

### Module not found errors
```bash
npm install
# or
rm -rf node_modules package-lock.json
npm install
```

### WebSocket won't connect
- Check backend WebSocket endpoint is running
- Verify `wsEndpoint` in settings matches backend
- Check browser console for CORS errors

### Authentication fails
- Verify backend API is running on port 8000
- Check CORS configuration in backend
- Verify JWT token handling

### Zustand persist errors
```bash
# Clear browser storage
localStorage.clear()
# Refresh page
```
